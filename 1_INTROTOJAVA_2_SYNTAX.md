### Operators and Casting
Arithmetic
|   name            |   sign    |   sample                              |
|---                |---        |                                    ---|                            
|   Addition        |   +       |   int a = 5 + 6;      // a is 11      |
|   Multiplication  |   *       |   int b = a * 7;      // b is 77      |
|   Subtraction     |   -       |   int c = b - a;      // c is 66      |
|   Division        |   /       |   double d = c / 6;   // d is 11.0    |
